# what is git ?
Git is a version control system (VCS). In simple terms,it is a time machine for yoyr code.
History sarvar : It tracks and saves every single change you make to your files.
Safety : If you make a mistake or accidentally break your code, you can easily go bake to an old working version.
Teamwork : Instead of emailing files back and forth, teams can use git to work on the exact same poaject without overwriting each other's work.

# How it works.
    Think of your project's progress like a video game.

## Day:1 : 
            You create a basic game and save it ( in git, this is called a commit ).
## Day:2 :
            You add a cool new feature and save it as a new version.
## Day:3 : 
            A bug appears! Instead of panicking or starting over, you use git to travel back to the version you saved on day ?

 # What is github?
    Github is an online platform where you can store ( save ) and manage your code.
    It is a paltform for software development that helps in these ways.

 # Git :
          Used to manage code on your local computer.
 # Github:
           Used to store and share that code on the internet.


 # Uses of Github :
                    - Saving projects online.
                    - Sharing code with others.
                    - Working together as a team.
                    
 # Key Features of Github. 
 
   - Feature                        - Description
   - Repository                     - A place to keep your files project.
   - Commit                         - Saving the changes you have made.
   - Branch                         - Creating different version of the project.
   - Pull Request                   - Merging changes into the main project.

 # Example Scenario :
                       - Suppose you are making a game.
