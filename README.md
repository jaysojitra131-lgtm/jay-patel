# what is git ?
Git is a version control system (VCS). In simple terms,it is a time machine for yoyr code.
History sarvar : It tracks and saves every single change you make to your files.
Safety : If you make a mistake or accidentally break your code, you can easily go bake to an old working version.
Teamwork : Instead of emailing files back and forth, teams can use git to work on the exact same poaject without overwriting each other's work.

# How it works.
    Think of your project's progress like a video game.

## Day:1 :

## Day:2 :
            You add a cool new feature and save it as a new version.
## Day:3 : 
            A bug appears! Instead of panicking or starting over, you use git to travel back to the version you saved on day ?

 # What is github?
    Github is an online platform where you can store ( save ) and manage your code.
    It is a paltform for software development that helps in these ways.
      - Keep your project code safe.
      - Share code easily with other people.
      - Many people can work on the same project together.
      - Suggest changes to other people's code. 

 # Git :
          Used to manage code on your local computer.
 # Github:
           Used to store and share that code on the internet.


 # Uses of Github :
                    - Saving projects online.
                    - Sharing code with others.
                    - Working together as a team.
                    
 # Key Features of Github. 
 
   - Feature                        - Description
   - Repository                     - A place to keep your files project.
   - Commit                         - Saving the changes you have made.
   - Branch                         - Creating different version of the project.
   - Pull Request                   - Merging changes into the main project.

 # Example Scenario :
                       - Suppose you are making a game.
                       - You save your code in git.
                       - Then, you upload it to github.
                       - Now, you can access it from any device and show it to your friends.

# What is the internet?
        - The internet is a huge network connecting computers and devices world wide.
        - Millions of people and machines are connected to each other through it.
        - It is a sustem through which information can be sent and received.

# How does the internet work?
        - Data ( information ) travels on the internet as follows.
        - Information is divided into small packets.
        - These packets reach from one device to another via cable, satellite, or wireless means.

# Example:
           - When you search for something on google.
           - Your request goes to the server via the internet.
           - The answer comes back from there to your mobile or laptop.
           - In this way, the internet connects millions of devices around the world.    




 # What is the intranet?
     An intranet is a private network that belongs to an organaization ( like a company,scholl or goverment office ) and is used only by its members.

 # Simple explanation :
                         - It works like the internet, but it's restricted to a specific group of people. 
                         - Only authorized users ( employees,students,etc..) can access it.
                         - Ii is usually protected by passwords or login systems.

 # What it's used for :
                         - Sharing internal information ( documents,announcements ).
                         - Communication ( emails,systems,project,management,databases).

 # Example :
             A company might have an internet where employes can.

              - Check their salary details.
